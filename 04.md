```bash
ls : 현재 디렉토리 확인
ls -l : 상세 목록 (권한, 소유자, 크기, 수정시간 등)
ls -a : 숨김 파일 까지 표시
ls -lh : 보기 쉬운 단위로
ls -R : 하위 디렉토리 재귀적 표시
ls -t : 최근 수정 순서 정렬
ls -S : 파일 크기 순서 정렬

cp file1 file2 : file1을 file2로 복사
cp -r dir1 dir2 : 디렉토리 전체 복사
cp -i file1 file2 : 덮어쓰기 전 확인
cp -p file1 file2 : 원본의 권한, 소유자, 시간 정보 유지
cp -v file1 file2 : 복사 과정 자세히 표시

rm file : 파일 삭제
rm -i file : 삭제 전 확인
rm -f file : 강제 삭제 확인 없이
rm -r dir : 디렉토리 내부 내용 삭제
rm -rf dir : 디렉토리 내부 내용 포함 삭제 확인 없이

mv file1 fiel2 : file1을 file2로 이름 변경
mv file1 dir/ : 해당 디렉토리로 파일 이동
mv -i file1 file2 : 덮어쓰기 전 확인
mv -f file1 file2 : 강제 덮어쓰기
mv -v file1 file2 : 이동 과정 자세히 표시

-i : 확인 하기
-f : 강제 실행
-v : 과정 표시
```
